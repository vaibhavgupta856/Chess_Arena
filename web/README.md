# ChessArena frontend

React + Vite + Three.js client for ChessArena.

Full project documentation (architecture, API, setup, deploy) lives in the repo root:

**[../README.md](../README.md)**

## Quick start

From the repo root, run the Go API on `:8080`, then:

```bash
npm install
npm run dev
```

`/api` is proxied to `http://localhost:8080` (see `vite.config.ts`).
